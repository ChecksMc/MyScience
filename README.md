# MyScience
Repository für das Myscience Projekt Amplidar.
Die Repository beinhaltet alle Dokumente, die bei dem Projekt erstellt wurden.

## Struktur
Die Repository ist in folgende Ordner unterteilt:
- `documents`: Alle Dokumente, wie z.B. die Email an das Ordnungsamt, die Präsentation und teil der Planung
- `src`: Alle Quellcodes
- `Simulation`: Die SUMo Simulation

## LiDAR Scan Ergebnisse
Die Ergebnisse sind nicht in der Repository, da sie zu groß sind.
